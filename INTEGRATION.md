### Table: `prompt_logs`

เก็บ prompt ที่ผู้ใช้ส่ง

| Column | Type | Description |
|--------|------|-------------|
| `id` | TEXT PK | UUID (auto-generated by worker) |

## API Endpoints

### `POST /api/prompt`

บันทึก prompt log

**Headers**
```
Content-Type: application/json
X-Api-Key: <your-api-key>
```

**Body**
```json
{
  "session_id": "ba6b4b4c-f8e3-4915-8d64-5de198e7a09d",
  "cwd": "C:\\Users\\project",
  "char_count": 42,
  "approx_tokens": 12,
  "prompt": "เนื้อหา prompt"
}
```

**Response (200)**
```json
{ "ok": true }
```

---

### `POST /api/usage`

Content-Type: application/json

## ตัวอย่างการเรียก

### PowerShell

```powershell
$WORKER_URL = "https://claude-prompt-logger.cloudflare-training3.workers.dev"
$API_KEY    = "<your-api-key>"

function Send-PromptLog {
    param([string]$SessionId, [string]$Cwd, [string]$Prompt)
    try {
        $body  = @{ session_id=$SessionId; cwd=$Cwd; char_count=$Prompt.Length; approx_tokens=[int][math]::Ceiling($Prompt.Length/3.5); prompt=$Prompt }
        $bytes = [System.Text.Encoding]::UTF8.GetBytes(($body | ConvertTo-Json -Compress))
        $req   = [System.Net.HttpWebRequest]::Create("$WORKER_URL/api/prompt")
        $req.Method = "POST"; $req.ContentType = "application/json"; $req.Timeout = 5000
        $req.ContentLength = $bytes.Length; $req.Headers.Add("X-Api-Key", $API_KEY)
        $s = $req.GetRequestStream(); $s.Write($bytes, 0, $bytes.Length); $s.Close()
        $req.GetResponse().Close()
    } catch { }
}

function Send-UsageLog {
    param([string]$SessionId, [string]$Model, [int]$InputTokens, [int]$OutputTokens, [int]$CacheCreate, [int]$CacheRead)
    try {
        $total = $InputTokens + $OutputTokens + $CacheCreate + $CacheRead
        $body  = @{ session_id=$SessionId; model=$Model; input_tokens=$InputTokens; output_tokens=$OutputTokens; cache_creation_input_tokens=$CacheCreate; cache_read_input_tokens=$CacheRead; total_tokens=$total }
        $bytes = [System.Text.Encoding]::UTF8.GetBytes(($body | ConvertTo-Json -Compress))
        $req   = [System.Net.HttpWebRequest]::Create("$WORKER_URL/api/usage")
        $req.Method = "POST"; $req.ContentType = "application/json"; $req.Timeout = 5000
        $req.ContentLength = $bytes.Length; $req.Headers.Add("X-Api-Key", $API_KEY)
        $s = $req.GetRequestStream(); $s.Write($bytes, 0, $bytes.Length); $s.Close()
        $req.GetResponse().Close()
    } catch { }
}
```

### TypeScript / JavaScript (fetch)

```typescript
const WORKER_URL = "https://claude-prompt-logger.cloudflare-training3.workers.dev";
const API_KEY    = "<your-api-key>";

async function sendPromptLog(data: {
  session_id: string;
  cwd: string;
  char_count: number;
  approx_tokens: number;
  prompt: string;
}) {
  await fetch(`${WORKER_URL}/api/prompt`, {
    method: "POST",
    headers: { "Content-Type": "application/json", "X-Api-Key": API_KEY },
    body: JSON.stringify(data),
  }).catch(() => {}); // silently swallow — never block main flow
}

async function sendUsageLog(data: {
  session_id: string;
  model: string;
  input_tokens: number;
  output_tokens: number;
  cache_creation_input_tokens: number;
  cache_read_input_tokens: number;
  total_tokens: number;
}) {
  await fetch(`${WORKER_URL}/api/usage`, {
    method: "POST",
    headers: { "Content-Type": "application/json", "X-Api-Key": API_KEY },
    body: JSON.stringify(data),
  }).catch(() => {});
}
```

### Python

```python
import requests

WORKER_URL = "https://claude-prompt-logger.cloudflare-training3.workers.dev"
API_KEY    = "<your-api-key>"
HEADERS    = {"Content-Type": "application/json", "X-Api-Key": API_KEY}

def send_prompt_log(session_id: str, cwd: str, prompt: str):
    try:
        requests.post(f"{WORKER_URL}/api/prompt", json={
            "session_id": session_id,
            "cwd": cwd,
            "char_count": len(prompt),
            "approx_tokens": int(len(prompt) / 3.5) + 1,
            "prompt": prompt,
        }, headers=HEADERS, timeout=5)
    except Exception:
        pass

def send_usage_log(session_id: str, model: str, input_tokens: int,
                   output_tokens: int, cache_create: int, cache_read: int):
    try:
        requests.post(f"{WORKER_URL}/api/usage", json={
            "session_id": session_id,
            "model": model,
            "input_tokens": input_tokens,
            "output_tokens": output_tokens,
            "cache_creation_input_tokens": cache_create,
            "cache_read_input_tokens": cache_read,
            "total_tokens": input_tokens + output_tokens + cache_create + cache_read,
        }, headers=HEADERS, timeout=5)
    except Exception:
        pass
```

## Error Responses

| Status | Meaning |
|--------|---------|
| `200 {"ok":true}` | สำเร็จ |
| `401 {"ok":false,"error":"Unauthorized"}` | API Key ผิด หรือไม่ได้ส่ง header |
| `400 {"ok":false,"error":"..."}` | Body ไม่ถูกต้อง |
| `404 {"ok":false,"error":"Not Found"}` | endpoint ไม่มี |

---

## หมายเหตุ

- `id` และ `logged_at` ถูก generate โดย worker อัตโนมัติ ไม่ต้องส่งมา
- `session_id` ควรเป็น UUID หรือ string ที่ unique ต่อ 1 การสนทนา
- ทุก error ใน helper function ควร swallow เงียบๆ เพื่อไม่ให้ block main flow
- `total_tokens` = `input + output + cache_creation + cache_read`
